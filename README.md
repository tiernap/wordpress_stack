# WordPress Stack

This project is to deploy a wordpress app stack in 1: Docker and 2: Openshift

## 1. Docker
This stack uses docker compose to deploy two containers: db & app
The app container is built using Dockerfile.

Requirements:

- Docker
- Docker compose

To deploy:

`docker-compose up -d`

The app endpoint is:

http://localhost:8080
